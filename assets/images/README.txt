Recommended local image structure for easy maintenance:

assets/images/home/banner-1440x720.jpg
assets/images/home/icon-01.jpg
assets/images/home/icon-02.jpg

assets/images/about/profile-666x815.jpg

assets/images/campaigns/<campaign-name>/banner-1440x380.jpg
assets/images/campaigns/<campaign-name>/image-01.jpg
assets/images/campaigns/<campaign-name>/image-02.jpg

You can replace the URL values in index.html collections/campaignData with these local paths.
Example:
"assets/images/campaigns/safer-everyday/banner-1440x380.jpg"
